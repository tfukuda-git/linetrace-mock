LINE Official Account Manager の権限を共有

Webhook URL の書き換えとチャネルシークレットの取得に必要です

所要 2分

手順

  1. LINE Official Account Manager にログイン
  2. 右上の「設定」→ 左メニュー「権限管理」
  3. 「メンバーを追加」をクリック
  4. 以下のURLを共有リンクとして発行し、下の入力欄に貼り付け

Linetraceに付与する権限：「運用担当者（メッセージ配信）」

Webhook URLの変更とチャネル情報の閲覧のみ行います。メッセージの送信・友だちリストの閲覧は行いません。設定完了後に権限を削除いただいてOKです。

権限の付与方法がわからない場合
スクリーンショット付きの手順書を用意しています。
PDF手順書をダウンロード →
または、画面共有で一緒に設定する（Zoom予約）→

よくある質問

パスワードを教える必要がありますか？

いいえ。各プラットフォームの正規の権限共有機能（招待・パートナー追加）を使います。パスワードやログイン情報をお預かりすることはありません。

設定後に権限を削除しても大丈夫ですか？

はい。LINE OAMとGTMの権限は、設定完了後に削除いただいて構いません。Meta BMのパートナー共有のみ、CAPI送信のために維持が必要です。

どのくらいで完了しますか？

権限共有の確認後、通常1〜2営業日で全設定を完了します。完了後にメールでお知らせし、ダッシュボードが自動で有効になります。

有料プラン移行後も代行してもらえますか？

トライアル期間中の初回設定は無料です。有料プラン移行後は、広告アカウントの追加やCAPI再設定などを月額プラン内でサポートします。
