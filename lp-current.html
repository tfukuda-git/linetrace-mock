コンバージョン、広告に返せていますか？

広告経由の友だち追加は計測できても、その後の LINE 内アクションや LINE 経由の Web 購入は広告に返らないのが、多くのアカウントで起きている計測ロスです。

なぜこの計測ロスが起きるのか
他社の LINE 計測ツールは LIFF（認証画面方式）や計測 URL（直リンク方式）が主流。LIFF は LINE アプリ内のウェブビュー上で動作するため、LP のブラウザ Cookie を読み出せません。計測 URL は LP 自体を経由しないため、そもそも広告クリック ID を Cookie に保存する機会がありません。
結果、広告の AI（自動入札）が「本当に買ってくれるユーザー」を学習できず、配信最適化が止まり、CPA が高騰します。

FAQ

よくあるご質問

ご導入前によくいただくご質問をまとめています。記載のない事項はお気軽にお問い合わせください。

既存の Meta Pixel・GA4 はそのままで使えますか？

はい。Linetrace は既存の購入イベントを上書きせず、独立したカスタムイベントとして送信するため、既存の計測数値を汚さず、広告の機械学習用データだけを安全に追加できます。レポートの数値も変わりません。

既存の LINE 拡張ツール（エルメ・Lステップ・サブスクライン等）と併用できますか？

はい、併用可能です。Linetrace は計測専用ツールのため、メッセージ配信機能や顧客管理機能はお使いのツールをそのままご利用いただけます。Webhook は中継して既存ツールに転送する設計です。

Safari の ITP で 7 日後に Cookie が消えるのでは？

Linetrace は first-party Cookie（顧客サイトのドメイン上）で動作するため、ITP の影響を最小限に抑えた設計です。詳しくは技術ドキュメントをご確認ください。

データの保存場所・セキュリティはどうなっていますか？

計測データは AWS（東京リージョン）に保存。PII（氏名・メール・電話）は一切取得しない設計で、保存するのは LINE UID と計測パラメータのみ。HTTPS / RLS 等で厳格に保護しています。

導入にどのくらい時間がかかりますか？

自分で設定する場合は最短 10 分。GTM タグ 1 行と Webhook URL の設定、CAPI 接続の 3 ステップで完了します。設定に不安がある場合は「おまかせ設定（無料）」をお選びください。アクセス権をお渡しいただくだけで、こちらで全設定を代行します。

トライアル期間中・終了後の解約はどうなりますか？

2 週間の無料トライアル中はいつでも解約可能、クレジットカード未登録のため自動課金もありません。トライアル後の契約期間も月単位で、最低契約期間はありません。

使わない月の広告アカウント連携費はどうなりますか？（季節・商材で運用が変わる場合）

ダッシュボードから広告アカウント連携ごとに一時停止できます。停止中はその連携の CAPI 送信が止まり、その月の連携費（¥3,000）は課金対象外になります（計測・ダッシュボード閲覧は継続）。基本料金 ¥10,000 は契約中は継続します。繁忙期だけ連携し、オフシーズンは停止する、といった季節運用に対応しています。
